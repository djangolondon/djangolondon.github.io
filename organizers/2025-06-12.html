🤖 Python 3.14 subinterpreters

✨ Feature freeze special exception ✨

Python 3.14 will gain concurrent.intepreters to run multiple Python interpreters in the same process.

Lower overhead than multiprocessing, better isolation and concurrency compared to threads. 

from concurrent import interpreters
interp = interpreters.create()
interp.call_in_thread(func)
