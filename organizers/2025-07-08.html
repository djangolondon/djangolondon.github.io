🔒 CSP support landed

Django 6.0 will ship support for setting Content Security Policy (CSP) headers.

Thanks to Rob Hudson and all reviewers! 


from django.utils.csp import CSP

SECURE_CSP = {
    "default-src": [CSP.SELF],
    "script-src": [CSP.SELF, CSP.NONCE],
    "img-src": [CSP.SELF, "https:"],
}
